## 說明
嘗試用`React`、`Tailwindcss`重新架構，配色沿用2020練習時製作的[個人網站](https://livehighvu06.github.io/chenyixian/)。<br /><br />

## 更新
2024/07 新增了以下功能：

- **React Router：** 實現多頁面應用，使得網站可以有更多結構化的頁面路由。
- **Framer Motion：** 引入動畫庫，使得頁面過渡更加流暢。
